## Tech Stack

- Next.js
- React
- Node.js

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/gift-recommender.git
cd gift-recommender
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This app is deployed on Vercel. Every push to the main branch automatically deploys.

## Usage

1. Enter the recipient's age
2. Select their gender
3. Enter their interests (comma-separated)
4. Set your budget
5. Click "Get Recommendations"
6. Browse the suggested gifts!
